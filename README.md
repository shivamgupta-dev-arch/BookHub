# BookHub

Full Stack Book Store Application

## Tech Stack

### Backend

* ASP.NET Core 8 Web API
* Clean Architecture
* Entity Framework Core
* SQL Server
* JWT Authentication
* FluentValidation
* MediatR

### Frontend

* React
* Redux Toolkit
* Material UI

### Cloud & DevOps

* AWS
* GitHub Actions
* Docker

## Project Structure

* BookHub.API
* BookHub.Application
* BookHub.Domain
* BookHub.Infrastructure
* BookHub.Persistence

## Roadmap

* Book Management
